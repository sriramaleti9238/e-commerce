About Mini E-Commerce Showcase

This is an e-commerce showcase module with both user-facing and admin functions. It demonstrates a modern Java Spring Boot application with a clean architecture.

Features Implemented

  • Public Product List Page (/products)

    Show product name, price, image thumbnail, and detail link

  • Product Detail Page (/product/id)

    Show full product info: image, name, description, price

  • Admin Login Page (/admin/login)

    Authenticate administrator (basic session or JWT)

  • Admin Product Management (/admin/products)

    View product list

    Create/Edit/Delete product entries

    Accessible only after login

Technical Details

  • Backend: Java 1.8, Spring Boot 3.2.6
  • Security: Spring Security 6.x
  • Database: PostgreSQL
  • Frontend: Thymeleaf 3.x, Bootstrap 5, JavaScript
  • Architecture: MVC with separation of concerns
